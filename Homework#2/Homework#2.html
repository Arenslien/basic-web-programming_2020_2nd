퍼셉트론

목차

  • 구조

  • 동작

  • 학습

  1. 구조

    2. 퍼셉트론의 구조는 아주 단순하다. 입력층과 출력층이라 부르는 2개의 층이 있는데, 입력층은 아무런 연산을 하지 않으므로 층의 개수를 셀 때 제외한다.
    따라서 퍼셉트론은 1개의 층이 있다고 말한다. 층을 구성하는 노드는 원으로 표시하며, 입력층에 있는 입력 노드 하나는 특징 벡터의 특징 하나에 해당한다.

  2. 동작

    3. 퍼셉트론의 동작은 단순하다. 입력층에 특징 벡터 x=(x1, x2, ..., xd)T가 들어오면, 서로 연결된 특징값과 가중치를 곱한 결과를 모두 더한다.
    이렇게 얻은 값 s를 활성화함수(activation function)에 입력으로 넣고 계산한다. 활성함수의 출력이 퍼셉트론의 최종 출력이 된다.

  3. 학습

    4. 학습은 목적함수 설계와 델타 규칙 유도로 이루어진다. 목적함수를 가중치 갱신 규칙에 적용하면 된다. 그리고 목적함수를 편미분하여 그레디언트를 구한다.
    구한 그레디언트를 가중치 갱신 규칙에 대입하면 학습을 위한 식이 만들어진다. 보통 퍼셉트론 학습 규칙으로서 델타 규칙이라고 말한다.
